VDI 云桌面平台 —— 功能介绍
📖 概述
VDI 云桌面平台是一套基于 Proxmox VE 虚拟化和 Apache Guacamole 远程桌面网关的轻量级桌面即服务（DaaS）解决方案。它提供图形化管理界面，让管理员可以快速批量创建、分配和管理 Windows/Linux 虚拟桌面，终端用户通过浏览器即可安全访问自己的桌面环境，无需安装任何客户端软件。

⚙️ 核心功能模块
1. 资源监控看板
实时显示 CPU、内存、存储 等宿主机资源使用率

统计运行中、已关机桌面数量及模板总数

存储池详情展示（已用/总计/剩余空间）

2. 桌面全生命周期管理
单个创建：指定模板、存储、克隆方式、IP 获取方式、安全模式

批量创建：上传 CSV 文件，一键批量部署数十个桌面

桌面列表：分页展示所有桌面，支持按状态筛选

修改连接：在线修改桌面 IP、系统用户名/密码、安全模式

删除桌面：销毁虚拟机并自动清理 Guacamole 连接和用户

3. 电源操作
支持对虚拟机进行开机、关机操作

状态实时同步（运行中/已关机）

4. 快照管理
创建快照：保存虚拟机当前状态

回滚快照：一键恢复到历史状态

删除快照：释放存储空间

5. 定时任务调度
支持定时开机、关机、销毁、创建快照

任务列表弹窗查看，支持删除未执行任务

自动时区转换（北京时间存储为 UTC），准时执行

6. 审计日志
记录所有管理操作（创建、删除、修改、开关机等）

包含操作者、目标、时间、IP 地址

7. 用户自助门户
用户使用分配的账号登录，只能看到自己的桌面

支持在线修改密码

点击桌面名称直接通过 Guacamole 连接

8. 安全与认证
管理员与用户分离，各自独立认证

支持 Guacamole 多种安全模式（any/rdp/nla/tls）

数据库密码哈希存储，兼容 Guacamole 1.6.0

场景	需求痛点	VDI 解决方案

软件开发/测试	多版本环境切换，隔离测试	为每个开发者分配独立虚拟机，支持定时销毁清理

临时项目/外包	短期人员需要临时环境	快速创建桌面，设置到期自动销毁，资源回收

📦 部署优势
一键生成脚本：运行 generate-scripts.sh 自动生成所有安装模块和配置文件

模块化安装：6 个独立脚本，可单独或全部安装，出错时可重试单个模块

国内源加速：自动配置阿里云 APT 源、PyPI 镜像，部署速度快

无客户端：终端用户只需浏览器，支持主流操作系统

低成本：全部使用开源组件（Proxmox、Guacamole、MySQL、Flask）

🛠️ 技术栈
层级	技术
虚拟化	Proxmox VE (API)
远程桌面网关	Apache Guacamole + guacd
数据库	MySQL 8.0 / MariaDB
后端框架	Python Flask
定时任务	APScheduler
Web 服务器	Nginx
容器化	Docker + Docker Compose
