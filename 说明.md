🔧 核心改进点

模块	改进内容

01-system.sh	强制替换阿里云 APT 源，避免官方源卡顿

02-docker.sh	多镜像源自动切换，防止下载失败

03-guacamole.sh	使用 Python 生成 docker-compose.yml，彻底解决密码特殊字符问题

04-database.sh	直接通过 Python 连接 MySQL，无需 docker exec 传递密码

05-vdi-web.sh	修正模板复制路径（cp -r .），增加循环启动验证（最多 30 次）

06-nginx.sh	自动配置反向代理，重载 Nginx 后立即可用



📋 使用流程

安装方案一

执行 bash setup.sh 脚本，会自动生成modules文件夹下的各模块安装脚本


将 app.py 和完整的 templates/ 目录复制到 /opt/vdi-deploy/



编辑 /opt/vdi-deploy/config.env，修改 Proxmox 地址、数据库密码等参数



执行 bash /opt/vdi-deploy/install.sh 选择 1 全部安装

安装方案二

各模块独立安装，如果安装更新过程卡死，结束安装，重新执行安装即可。

安装方案三

已经生成各模块的安装脚本，和对应的html文件以及配套文件

进入 vdi-deploy下执行bash insall.sh 进行安装



等待安装完成，访问 http://<GUAC\_HOST>/ 管理面板
